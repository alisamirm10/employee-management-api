# Employee Management Dashboard API

Backend REST API for managing employees with authentication and role-based authorization.

## Features

- User Registration & Login
- Password Hashing with bcrypt
- JWT Authentication
- Role-Based Authorization
- Access Tokens
- Refresh Tokens
- Logout
- Employee CRUD Operations
- Pagination
- Filtering
- Search
- Sorting
- Request Validation
- Global Error Handling

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

## Installation
